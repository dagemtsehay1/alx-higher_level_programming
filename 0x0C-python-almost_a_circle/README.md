# Python - Almost a circle

  > In this project, we review everything about Python:
  >   - Import
  >   - Exceptions
  >   - Class
  >   - Private attribute
  >   - Getter/Setter
  >   - Class method
  >   - Static method
  >   - Inheritance
  >   - Unittest
  >   - Read/Write file
  >   
  > We also learn about:
  >   - args and kwargs
  >   - Serialization/Deserialization
  >   - JSON
